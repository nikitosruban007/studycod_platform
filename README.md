# StudyCod

StudyCod — це освітня платформа для навчання програмуванню з використанням штучного інтелекту, орієнтована на підвищення якості практичної підготовки учнів та студентів.

Проєкт створений як відповідь на проблему формального підходу до навчання програмуванню, коли основна увага приділяється запам’ятовуванню синтаксису, а не розвитку алгоритмічного та інженерного мислення.  
StudyCod переносить акцент з теоретичних пояснень на практичну роботу з кодом, його виконання, тестування та аналіз.

Платформа використовує штучний інтелект для автоматичної генерації навчальних матеріалів, практичних завдань, тестових даних і пояснень рішень. Це дозволяє адаптувати навчальний процес до рівня підготовки користувача та зменшити навантаження на викладача.

StudyCod підтримує два режими роботи: персональне навчання та освітній режим для організованого навчального процесу з класами, контрольними роботами та системою оцінювання.  
Автоматизована перевірка програмного коду мовами Java та Python забезпечує об’єктивність оцінювання та прозорість результатів.

Проєкт може використовуватися як у шкільній освіті, так і в позашкільних та дистанційних форматах, а також як основа для подальших досліджень у галузі застосування штучного інтелекту в освітніх системах.
